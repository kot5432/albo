# 挑戦アプリ プロトタイプ

ユーザーが初動する心理トリガーを体験できるプロトタイプアプリケーションです。

## 特徴

- **心理トリガー設計**: 24時間以内の初動要求、不可逆宣言、最小行動提案
- **ダークネイビーテーマ**: 集中できる静かなUIデザイン
- **モーダルベース**: ステップバイステップの操作体験
- **ローカルストレージ**: データはブラウザに保存

## 画面構成

1. **LP (Landing Page)**
   - ロゴと「挑戦を始める」ボタン
   - ミニマルな開始画面

2. **宣言作成モーダル**
   - 具体的な挑戦内容入力
   - AIによる抽象表現チェック
   - 期限設定

3. **AI初動提示モーダル**
   - 今日やる最小行動をAI提案
   - 編集可能なアクションプラン

4. **不可逆確認モーダル**
   - 宣言の不可逆性警告
   - 理解確認チェックボックス

5. **ホーム画面**
   - 挑戦カード表示
   - 24時間カウントダウン
   - 今日の行動表示
   - 他人の挑戦人数表示

6. **記録モーダル**
   - テキスト記録入力
   - 画像添付機能
   - 編集不可（追記のみ）

7. **挑戦詳細モーダル**
   - 挑戦情報表示
   - 行動ログ
   - 再挑戦機能

8. **履歴画面**
   - 統計表示
   - 過去挑戦リスト

9. **構想ノート**
   - 自由なアイデア記述
   - 挑戦への変換機能

10. **挑戦理由モーダル**
    - 理由入力
    - 他人の理由匿名表示

## 技術仕様

- **HTML5**: セマンティックマークアップ
- **CSS3**: モダンなスタイリング、レスポンシブ対応
- **Vanilla JavaScript**: 状態管理、モーダル制御、カウントダウン
- **LocalStorage**: データ永続化

## 心理設計ポイント

- **初動24時間**: 心理的ハードルを下げる時間制限
- **不可逆宣言**: 失敗しても履歴に残る責任感
- **最小行動**: 「今日やること」の具体化
- **匿名性**: 他人との比較による動機付け
- **シンプルUI**: 集中を妨げない最小限のデザイン

## 使用方法

1. `index.html` をブラウザで開く
2. 「挑戦を始める」をクリック
3. 宣言を作成し、AI提案を確認
4. 不可逆性を理解して挑戦開始
5. ホーム画面でカウントダウンを確認
6. 記録機能で進捗を管理
